## ✨ Purpose

This tool was built to:

- Standardize support posts across **X, Instagram, Threads, and Facebook**
- Maintain consistent **luxury-aligned language** when posting about Dior
- Support campaign visibility through **clean, organic caption variations**
- Reduce repetitive spam captions across timelines
- Generate thousands of natural-sounding fashion captions quickly
- Streamline coordinated posting using **Copy Caption** / **Copy Body** tools

This is a **static tool**. No data is collected and no captions are stored remotely.

---

## 🖤 Core Principle

All generated captions aim to:

- Maintain alignment with **Dior’s luxury aesthetic**
- Keep the tone **editorial, refined, and fashion-focused**
- Support **LingOrm as a unit representing Dior**
- Avoid repetitive fan-style captions
- Produce captions that read naturally in fashion conversations

## 🧭 Platform Modes

The generator supports four posting environments.

### 1️⃣ X

- Outputs caption text followed by the campaign tag block
- Preserves clean spacing between caption and tags
- Optimized for fast copy-paste posting

### 2️⃣ Instagram

#### IG Comments
Outputs **caption text only**, without hashtags or mentions.

Best for:
- comment boosting
- clean reply chains
- algorithm-friendly engagement posts

#### IG Story
Outputs caption text **plus the Instagram tag block**.

### 3️⃣ Threads

Outputs caption text followed by the Threads tag block.

Designed for:
- replies
- quotes
- conversation posts

### 4️⃣ Facebook

Outputs caption text followed by the Facebook tag block.

- Editable hashtag blocks inside the settings panel
- Fast generation of multiple captions per batch
- Scroll-friendly layout for mobile posting sessions

---

## 📋 Copy Tools

Each caption includes flexible copy options.

**Copy Caption**
- Copies the full generated output, including caption and tag block

**Copy Body**
- Copies the caption text only

**Tap-to-Copy**
- Clicking directly on the caption text copies the full output

This allows users to adapt captions quickly for different platforms.
